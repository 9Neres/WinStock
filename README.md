<h1>App de Inventário com Leitor de Código de Barras</h1>

<p>
Aplicativo Android de inventário com leitura de código de barras, registro local e sincronização com backend via API REST, seguindo um fluxo offline-first. [web:14][web:8]
</p>

<p>
Stacks: Kotlin, Jetpack Compose, Android SDK (API 24+), CameraX, ML Kit Barcode Scanning, Zebra DataWedge, Ktor Client, Kotlinx Serialization, Coroutines, SharedPreferences / StateFlow, NocoDB (REST API). [web:7][web:1]
</p>

<p>
O projeto é focado em boas práticas de arquitetura mobile, consumo de API, cache offline e integração com scanners industriais, usando IA como apoio em código e documentação. [web:14][web:18]
</p>
